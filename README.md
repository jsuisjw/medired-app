# Medired App

## Project Documentation

The Medired App is a user-friendly application designed for managing health-related information efficiently. This project aims to provide users with a seamless experience in logging and accessing their medical records, medications, and health statistics.

## Features
- **User Authentication**: Secure login and registration process.
- **Health Record Management**: Users can log and manage their medical records.
- **Medication Tracking**: Keep track of medications, dosages, and schedules.
- **Health Statistics**: Visual representation of health data over time.
- **Reminders**: Set reminders for medication and appointments.
- **Data Security**: All data is encrypted and stored securely.

## Installation Instructions
1. **Clone the repository**: 
   ```bash
   git clone https://github.com/jsuisjw/medired-app.git
   ```
2. **Navigate into the project directory**:
   ```bash
   cd medired-app
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Set up environment variables**: Create a `.env` file in the root directory and add the necessary environment variables as per `.env.example`.
5. **Start the application**:
   ```bash
   npm start
   ```

## Usage Guide
- After setting up the application, navigate to `http://localhost:3000` in your web browser.
- Register a new account or log in with your existing credentials.
- Access the dashboard to begin managing your health records and medications.
